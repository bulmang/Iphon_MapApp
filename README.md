# SwiftUI Map App - Toy Project

![MapApp](https://user-images.githubusercontent.com/114594496/227917806-46cd990a-ccc1-4bff-b1fe-c07289bcdd36.gif)

## 앱 설명

- 아이폰 내부 지도 api를 이용하여 지도를 구현 해봤습니다.

## 앱 기능

- 유저 위치 권한 허락 설정
- 검색 후 검색 리스트 누르면 위치 이동
- 현재 내 위치 표시 버튼
- 위성 , 그림 지도 전환 버튼
- 검색 한 곳 마커 찍기
